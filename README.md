# Python Learning & Project Workspace

Welcome to a personal Python workspace containing tutorials, mini-projects, and experiments across several domains.

## 🚀 Overview
This workspace is organized into multiple folders that cover:
- Python fundamentals and chapter-based practice
- Advanced Python topics and utilities
- OOP examples
- CLI and voice assistant projects
- Chat bot / Discord bot work
- Django tutorial project
- SQLite student database demo

## 📁 Workspace Structure

### `advance_python/`
Advanced Python concepts and small applications, including:
- `args_and_kwargs.py` — argument handling examples
- `grade_evaluator.py` — grade calculation logic
- `number_guess.py` / `guessnumber/` — guessing game variants
- `bank_mgnt_system/` — simple bank management system
- `magic_method/`, `numpy/`, `pandas/` — demonstrations of Python internals and data libraries
- `student mgnt/` — student records management scripts

### `AI-Chat-bot/`
Contains a Discord-based bot project and chat bot scripts:
- `discord_bot.py` — bot implementation for Discord
- `position_check.py` — likely helper logic or bot utility

### `Django-tuto/`
A Django tutorial project scaffold:
- `mysite/` — Django project with app configuration, database, and templates
- `db.sqlite3` — SQLite database file
- `filerun.txt` — notes or tutorial instructions

### `Jarvis/`
A voice assistant project with AI and speech capabilities:
- `main.py` — core application entrypoint
- `ai_module.py` — OpenAI and AI response handling
- `musicLibrary.py` — music playback features
- `news.py` — news fetching utilities
- `requirement.txt` — dependency list
- Includes a dedicated `README.md` with usage notes

### `oop_concept/`
Object-oriented programming examples:
- `001_basic oops_concept.py`
- `002_inheritance_ploymorphism.py`
- `003_abs-and-mixin.py`

### `python/`
A chapter-based Python practice folder:
- `chap1.py` through `chap13.py` — learning exercises
- `perfect_guess.py`, `snake-water-gun.py` — game scripts
- `donkey.txt`, `poem.txt`, `table.txt` — text assets for practice
- `requirements.txt` — dependency list for this folder
- Nested `python/` folder appears to contain a copy of the same chapter practice files

### `python_db_sqlite/`
SQLite database tutorial example:
- `student_db.py` — database operations and student records
- `school.db` — SQLite database file
- `README.md` — local documentation for the SQLite demo

## 🛠️ How to Use This Workspace

1. Explore the folder that matches your interest.
2. Open relevant Python files and run them with Python 3.x.
3. Install dependencies from `requirements.txt` when available.
4. For Django, use `manage.py` commands inside `Django-tuto/mysite/`.

## 📌 Notes
- Some folders contain tutorial-style content and example projects rather than production-ready applications.
- The `Jarvis/` project is the most feature-rich assistant demo in this workspace.
- The `python/` folder contains duplicate content inside a nested `python/` subfolder.
